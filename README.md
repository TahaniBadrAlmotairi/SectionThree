## Getting Started

استعملت في الصفحة عدة عناصر من Flutter، أهمها:
•	Scaffold لبناء الهيكل الأساسي.
•	AppBar لعرض العنوان.
•	SafeArea لحماية المحتوى من الحواف.
•	SingleChildScrollView للتمرير.
•	Column + Padding + SizedBox لترتيب وتنظيم المحتوى.
•	Stack + Positioned لوضع عناصر فوق بعض.
•	Container لعرض مربعات وعناصر القائمة.
•	ListView.builder لإنشاء قائمة ديناميكية.
•	FloatingActionButton لإضافة عنصر جديد.
•	Obx + Get.put من GetX لإدارة الحالة وتحديث الواجهة تلقائياً.
•	Splash Screen مع تايمر لعرض شاشة البداية قبل الانتقال للصفحة الرئيسية.
•	تغيير أيقونة التطبيق باستخدام flutter_launcher_icons
